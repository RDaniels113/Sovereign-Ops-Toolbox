# Sovereign Ops Toolbox Roadmap

## Purpose

Sovereign Ops Toolbox is a growing collection of security, infrastructure, and operational automation scripts designed to support real-world IT administration and security operations.

The focus is on practical tools that improve visibility, consistency, and operational efficiency.

---

## Current Focus Areas

### Endpoint Management

* Endpoint compliance reporting
* System inventory collection
* Windows configuration validation
* Baseline verification

### Microsoft 365

* License reporting
* User inventory reporting
* Administrative health checks
* Service configuration validation

### Entra ID

* User auditing
* MFA status reporting
* Conditional Access review
* Role assignment validation

### Infrastructure Operations

* System health checks
* Server inventory reporting
* Configuration auditing
* Operational diagnostics

### Security Operations

* Security baseline validation
* Event log collection
* Configuration auditing
* Incident response support tooling
