# Security Policy

## Supported Versions

This repository is maintained on a best-effort basis.

Security updates and fixes will be applied to the most current version of scripts and documentation when appropriate.

| Version | Supported |
| ------- | --------- |
| Current | Yes       |
| Legacy  | No        |

---

## Reporting a Vulnerability

If you discover a security issue within this repository, please report it responsibly.

Please include:

* A description of the issue
* Steps to reproduce
* Potential impact
* Suggested remediation (if known)

Reports may be submitted through:

* GitHub Issues (for non-sensitive findings)
* Direct contact through GitHub for sensitive disclosures

Please do not publicly disclose vulnerabilities until they have been reviewed.

---

## Scope

This repository contains scripts, documentation, and operational tooling intended for educational and administrative purposes.

Users are responsible for validating all scripts and configurations in a test environment prior to production use.

No warranty is provided regarding fitness for any specific environment or use case.
